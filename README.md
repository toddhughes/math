# Math

A personal math learning project. Each topic is a folder of Markdown
files written as a read-along book: intuition first, then worked
examples, then the formal idea.

## Topics

- [Calculus](calculus/00-introduction.md)
- [Linear Algebra](linear-algebra/00-introduction.md)

More topics will be added over time.

## How to use this

1. Open the topic folder you want to study.
2. Read its `00-introduction.md`. That is the book's landing page
   for the topic. For large topics, it links to several **parts**,
   each of which has its own sub-folder and its own introduction.
3. Read the chapters in order. Each chapter ends with a link to the
   next.

## Chapter availability

The chapter list lives in the `00-introduction.md` of whichever
container holds the chapters -- the part folder when a topic is
split into parts, or the topic folder when it is not. The marker
next to each chapter shows whether it has been written:

- `[x]` written and available to read
- `[ ]` planned, not yet written

Chapters marked `[ ]` appear in the table of contents without a link.
They become linked once written.

## Math notation

Lessons use LaTeX-style math: `$...$` for inline and `$$...$$` for
display. To see the math rendered, open the files in a renderer that
supports it -- GitHub, VS Code with a Markdown+Math preview extension,
or Obsidian all work well.
